# PRODIGY_DS_01
# Population Data Analysis and Visualization

## Project Overview

This project focuses on analyzing population data using data visualization techniques. The objective is to understand the distribution of categorical and continuous variables within a population dataset.

The project demonstrates how graphical representations such as bar charts and histograms can be used to identify trends and patterns in demographic data.

---

## Dataset Description

The dataset contains information about 100 individuals with the following attributes:

| Column Name | Description |
|------------|-------------|
| Person_ID | Unique identifier for each individual |
| Age | Age of the individual |
| Gender | Gender category of the individual |

### Variable Types

| Variable | Type |
|----------|------|
| Gender | Categorical Variable |
| Age | Continuous Variable |

---

## Project Objectives

- Analyze demographic data.
- Visualize categorical variables using bar charts.
- Visualize continuous variables using histograms.
- Understand population distribution patterns.
- Gain insights from graphical analysis.
